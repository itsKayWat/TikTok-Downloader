Privacy Policy for TikTok Downloader

Effective Date: 02/12/2025

Introduction

TikTok Downloader is a browser extension designed to help users download TikTok videos and audio for personal use. The extension automatically detects TikTok videos on the page and provides simple download options through an intuitive interface.

Data Storage

This extension stores only the following data temporarily in your browser:

  • Current video URL for download
  • Video title for filename generation
  • Download preferences (video/audio format)

All data is processed locally and temporarily. No permanent storage is used, and data is cleared when you close the tab.

Data Collection

TikTok Downloader operates with complete privacy:

  • No personal information is collected
  • No user data is stored permanently
  • No analytics or tracking is performed
  • Only processes current page video information

Data Usage

The temporary data is used solely for:

  • Detecting available videos on the current page
  • Processing download requests
  • Generating appropriate filenames
  • Managing download formats (video/audio)

Data Sharing

We do not share any user data with third parties. All video processing and downloads are handled directly between your browser and TikTok's servers.

User Rights

Users have complete control over the extension:

  • Toggle video detection on/off
  • Choose download format (video/audio)
  • Select download location
  • Control when downloads occur
  • Disable the extension at any time

Contact Information

If you have any questions or concerns about this privacy policy, please reach out to us on TikTok or GitHub.
